Types of Plug-Ins

The following types of plug-ins are available.

Plug-In Type Plug-In Name Description
File Format Plug-Ins NW4C_Tga The TGA format plug-in.
Saves and loads Targa (TGA) files that have additional information for NintendoWare.
NW4C_ctex The CTEX format plug-in.
Saves and loads 3D texture intermediate files (CTEX files).
Filter Plug-Ins NW4C_ColorDepth A color depth filter plug-in.
This filter converts the color depth to correspond to the format.
NW4C_ETC An ETC filter plug-in.
This filter applies an ETC compression format to what is displayed
NW4C_NormalMapFilter A normal map filter plug-in.
This filter creates a normal map based on a height map.
Automatic Process Plug-Ins NW4C_CreateMipMap The Mipmap Creation plug-in.
Automatically generates image data for each level of the mipmap.
NW4C_CollectParticleImages The Collect Particle Images plug-in.
This plug-in automatically collects particle images in the specified folder into one image.


Note: NW4R_General, which is included with the plug-ins, is a filter plug-in used by the other plug-ins. The user cannot run this plug-in directly.
