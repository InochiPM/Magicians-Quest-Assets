CTEX Format Plug-In

This plug-in saves and loads 3D intermediate texture files (CTEX files).

Saving
Loading
Notes

Saving

To save a texture as a 3D intermediate file, select File > Save as, and set the file format to NW4C_ctex.
Enter a name for the CTEX file and click Save. A dialog box appears.

The options are the same as for the TGA Format plug-in.
However, you cannot select the For LayoutEditor option.

This plug-in can save CTEX files conforming to version 0.7.0.


Loading

To load a 3D intermediate file for a texture (a CTEX file), select File > Open, select the CTEX file, and then click Open.
You can also load CTEX files by dragging them to the Photoshop window.

Notes

Notes are the same as for the TGA Format plug-in.
Note, however, that the image quality will not be retained when you reload a saved NW4C_ctex file if the format was something other than L8, A8, LA8, HILO8, RGB8, or RGBA8.
The size for both the image height and width must be a power of two between 8 and 1024.

About the Path attribute of the <ImageTextureCtr> tab in the 3D intermediate file (CTEX)
The Path attribute shows the path of the original file.
If you selected File > Save As to save the CTEX file, the original filename (that was loaded in PhotoShop) is saved here, instead of the name of the saved CTEX file.
