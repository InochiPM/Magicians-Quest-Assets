Notes on the ETC Format

The process of saving NW4C_Tga (NW4C_ctex) files differs, depending on whether the ETC filter has been applied to the image.
The differences are described below.

When the ETC Filter Has Been Applied
When the ETC Filter Has Not Been Applied

When the ETC Filter Has Been Applied

If you apply the ETC filter to an image and then do nothing else and save this as an ETC format NW4C_Tga (NW4C_ctex) file, the data exported to the TGA file as additional information and to the CTEX file is exported in this post-filtered state without double compression.

If an ETC filter has been applied and then data is saved, the Photoshop image quality cannot be changed back to the state before the filter was applied. When applying filters, be careful not to save the resulting data in the same file used prior to filtering.


If the image is modified after applying the ETC filter, that image is handled in the same manner as if the filter was not applied.

When the ETC Filter Has Not Been Applied

If you do not apply the ETC filter to an image and save it as an ETC format NW4C_Tga (NW4C_ctex) file, the data exported to the TGA file as additional information and to the CTEX file is data for the compressed state of the image when saved.
The image quality in Photoshop is always for the image in its pre-compressed state, so there is no degradation in image quality when the NW4C_Tga file is saved.
