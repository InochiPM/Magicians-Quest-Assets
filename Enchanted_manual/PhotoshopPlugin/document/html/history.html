Revision History

2010-10-20

Additions and Changes

  • Added support for Adobe Photoshop CS5 (32-bit/64-bit).

Bug Fixes

  • Revised the Mipmap Creation plug-in so that the display of ALL(ETC1) would not be cut off

2010-09-20

Additions and Changes

  • In the Save dialog box for TGA files, the For LayoutEditor option is now a choice instead of a compulsory selection when a texture is not the right size or is a cube map texture.
  • The selection made for For LayoutEditor option in the Save dialog box for TGA files is now saved to and read from the registry.
  • When selecting the ETC1 compression format in the dialog boxes to save TGA and CTEX files, all data is cached after it has been encoded.
  • Added ALL(ETC1) (the maximum number of ETC1 compressed mipmaps) to the list in the Mipmap Creation plug-in.

2010-07-16

Additions and Changes

  • Added 64-bit support.
  • The normal map creation plug-in now processes by layers.
  • Added a feature to the normal map creation plug-in for multiplying the alpha channel by the height map.
  • Added a feature to the normal map creation plug-in for saving and loading settings.
  • The folder selection dialog box can now be resized when using the plug-in for collecting particle images.

Bug Fixes

  • Fixed a bug in the Normal Map Creation plug-in where the alpha channel would not be not multiplied if there was only a background layer and not all of the channels—RGB and alpha—were selected.
  • Fixed a bug with the Normal Map Creation plug-in where a file extension was not being automatically added when saving settings.

2010-06-22

Additions and Changes

  • Updated the plug-in to version 0.8.0.
  • When an 8x8 pixel mipmap is being created, you can now select ETC1 on the NW4C_Tga Format Option screen, and when you go to save the file, a message dialog box now appears.
  • Changed the processing with the HILO8 format so that the B component is set to 0x00 and the A component is set to 0xFF.
  • Updated the CTEX version information to version 0.7.0.
  • Added a plug-in for creating normal maps.
  • Added a plug-in for collecting particle images.

2009-12-10

Additions and Changes

  • Updated the plug-in to version 0.3.0.

Bug Fixes

  • Fixed a bug where data could not be saved in LA4 format when the image width or height was 8.
  • Fixed a bug where data in ETC1_A4 format was not being saved correctly.
  • Fixed a bug where the ETC filter did not work correctly when the data was in the ETC1_A4 format.

2009-11-10

Additions and Changes

  • Updated the plug-in to version 0.2.1.
  • The File Format plug-in now supports the L4 and A4 formats.
  • The Color Depth Filter plug-in now supports the L4 and A4 formats.

2009-10-30

Initial version.
