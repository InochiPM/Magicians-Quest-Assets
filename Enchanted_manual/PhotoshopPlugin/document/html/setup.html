Installing the Photoshop Plug-Ins

Follow this procedure when manually setting up the Photoshop plug-ins.

If you are using Installer to install or uninstall the plug-ins, the following procedure is not necessary.
(At this time, no installer has been prepared.)

Installation Procedure

Exit Photoshop

Exit Photoshop if it is currently running.
Exit Adobe Bridge if it is currently running.

Copying Files

Copy the NW4C_Photoshop_Plugin folder (located in (NintendoWare for CTR_installation_folder)\tools\DccPlugin\Photoshop) to one of the following locations.

・For Photoshop CS2:
(Photoshop_installation_folder)\Plug-ins\Adobe Photoshop Only\

・For Photoshop CS3:
(Photoshop_installation_folder)\Plug-ins

・For Photoshop CS4:
(Photoshop_installation_folder)\Plug-ins

・For Photoshop CS5:
Photoshop_installation_folder)\Plug-ins

Copying Files (64-Bit Version)

Copy the NW4C_Photoshop_Plugin_64 folder (located in (NintendoWare for CTR_installation_folder)\tools\DccPlugin\Photoshop) to one of the following locations.

・For Photoshop CS4 (64 bit):
Photoshop_installation_folder)\Plug-ins

・For Photoshop CS5 (64 bit):
Photoshop_installation_folder)\Plug-ins


If Adobe Bridge is installed, copy NW4C_ctex.8bi from the (NintendoWare for CTR_installation_folder)\tools\DccPlugin\Photoshop\NW4C_Photoshop_Plugin folder to the following location. CTEX file thumbnails can be displayed when using Adobe Bridge.

(Adobe Bridge installation folder)\Plug-Ins\


This completes the installation.

If the plug-ins have been installed correctly, NintendoWare for CTR is added to Filter menu of Photoshop.
To open the manuals for the plug-ins, select Help > About Plug-in > NW4C_xxxx (for example NW4C_Tga). A PhotoShop dialog box appears. In the dialog box, click Help Index.

Uninstalling

Exit Photoshop if it is running, and then delete all folders that you copied to the NW4C_Photoshop_Plugin folder (or to the NW4C_Photoshop_Plugin_64 folder) at installation time.
If Adobe Bridge is running, exit and delete the NW4C_ctex.8bi file that was copied during installation.
This completes the uninstall process.
